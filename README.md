# Claude Code Plugins by Patrick Chouinard

A personal marketplace for Claude Code plugins.

## Plugins

| Plugin | Description |
|--------|-------------|
| [cc-statusline](https://github.com/hopchouinard/CC-StatusLine) | 4-line ANSI dashboard statusline showing environment, context window, session cost, and git status |

## Usage

Add this marketplace to Claude Code:

```
/plugin marketplace add hopchouinard/claude-marketplace
```

Then install any plugin:

```
/plugin install cc-statusline@hopchouinard-plugins
```
